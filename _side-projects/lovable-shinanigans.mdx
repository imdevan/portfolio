    # lovely.sh
  </FIP>

  <FIP>
    I made a script to easily get lovable pages ready to deploy to a github page.
  </FIP>

  <FIP>
    The script essentially tweaks the config and Browser Router to account for non-root github pages

    I personally keep the script in my dot files which is called triggering the script in a lovable generated repo.

    ```
    ~/dotfiles/functions/lovable.sh
    ```

    The script will generate a `.github/workflows/deploy` script, update the `vite.config.ts`, `readme.md`, and `src/App.tsx`

    I of course have an alias mapped to the script, so I just call

    ```
    lov
    ```

    from the lovable directory

  </FIP>

    <p className="text-center">
